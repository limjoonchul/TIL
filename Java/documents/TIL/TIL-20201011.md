아직도 활용방법에 대해 헷갈리고 이론적인 부분으로도 헷갈리는 것 같다. 다른 블로그 정리된거 찾아보면서 더 보강을 해야겠다.

* 부족하다가 생각했던 부분에 대해서 강의를 들으면서 정리를 하긴 했지만, 아직 완벽히 정리된 것 같지 않다.
다음주부터 학원을 다시 나가게 되고, 프로젝트를 진행하게 되는데 3개월 가까이 자바 기초에 대해서 공부하는 시간을 가졌지만
온전히 내것으로 만들지 못했다는 점과 이런 상태에서 프로젝트를 진행하게되고, Spring 강의를 진행해야 한다는 점이
걱정이 많이 된다. Spring을 나가게 되면 실질적으로 자바에 대한 공부할 시간이 없어질테고, 그날그날 배운거에 대해서
공부하는 것만으로도 시간이 많이 부족할테니깐.. 열심히 해보자 아직 난 진짜 열심히 무언가를 해본적이 없는 것 같다.
